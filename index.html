In the original study, this page directed students to a questionnaire relating to their attitudes and emotions towards debugging. For this version, click the button below to view and attempt the exercises again.

There are 5 programming exercises to complete which, will be displayed one at a time below. Once you’ve finished, press the NEXT EXERCISE button at the bottom of the page. But beware- once you’ve pressed next you can’t go back!

To help you find all of the errors, make sure you test your code with a range of values using the pink run button.

These exercises are not assessed, so don't worry if you are struggling to solve all of the errors. Try your best and feel free to move on if you are finding an exercise too difficult.
